AnchorLinkGDImage

GDImage is a simple extended library based on PHP Image Processing and GD for easy image handling.
With GDImage you can read, handle, create and save your images easy.

GDImage runs with PHP 5.3 or higher and requires PHP GD library.

AnchorLink Getting started

GDImage Library provides a four classes: GDImage (Parent class), GDFigure, GDText and GDCanvas.

  • With GDImage class you can handle some image (JPG, PNG or GIF), for example you can load an image from local or external url, resize and then save in PNG format.
  • With GDFigure class you can create some figure on solid RGB color.
  • With GDText class you can create some string text, for example create some text based on your favorite TTF font.
  • With GDCanvas class you can add several objects for create a unique image. For example you can create a banner with some local or external photos.
